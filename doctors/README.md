# Doctors Plugin

Плагин добавляет кастомный тип записей **Doctors** с таксономиями, метаполями и фильтрацией.

## Установка и запуск
1. Скопируйте папку `doctors` в `wp-content/plugins/` и активируйте плагин в админке WordPress.  
2. Загрузите демо-данные через WordPress Importer, используя файл `demo.xml`. 

## Почему `pre_get_posts`

Использование `pre_get_posts` позволяет:  
- Изменять основной запрос архива CPT без правок шаблонов темы  
- Корректно работать с пагинацией  
- Применять фильтры и сортировку через GET-параметры  

Для локальных выборок (например, в шорткодах или виджетах) можно использовать аккуратный `WP_Query`.
